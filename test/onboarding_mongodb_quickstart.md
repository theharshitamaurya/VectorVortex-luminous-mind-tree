Onboarding: MongoDB Quickstart
Section 1 — Local Setup: install MongoDB Community via brew/apt/winget, start mongod, connect with mongosh. Section 2 — Atlas Setup: create free M0 cluster, whitelist IP, create DB user, get connection string. Section 3 — CRUD with mongosh: insertOne, insertMany, find with query filters, updateOne with $set, deleteOne. Section 4 — Using a Driver (Node.js example): install mongodb npm package, connect with MongoClient, perform async CRUD. Section 5 — Data Modeling basics: embedding vs. referencing, document size limit (16MB), schema design patterns (bucket, outlier). Section 6 — Indexes: createIndex, explain plan, compound indexes, index on arrays. Section 7 — Key resources: MongoDB University free courses, official docs, Atlas UI tour, community forums.
