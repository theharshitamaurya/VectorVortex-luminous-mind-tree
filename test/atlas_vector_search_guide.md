Atlas Vector Search Guide
Explain vector embeddings and why Atlas lets you colocate them with operational data (no separate vector DB needed). Cover prerequisites: Atlas M10+ or Serverless, MongoDB 6.0.11+ or 7.0.2+. Walk through creating a vector search index (fields: type "vector", path, numDimensions, similarity — cosine/euclidean/dotProduct). Show how to insert documents with embeddings. Demonstrate the $vectorSearch aggregation stage with numCandidates and limit. Cover filtered vector search for combining semantic + metadata filters. Include a RAG pipeline pattern: chunk → embed → store → query → retrieve context → pass to LLM. Performance tips: set numCandidates to 10–20x your limit; use M30+ for production; consider hybrid search combining $vectorSearch with $search (BM25).
